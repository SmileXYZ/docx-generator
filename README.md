# DocGen — Генерация Word-документов по шаблону

Веб-приложение для автоматической генерации `.docx` документов по загруженным шаблонам с плейсхолдерами `{{поле}}`.

## Возможности

- 📄 Загрузка `.docx` шаблонов с плейсхолдерами `{{field}}`
- 🔍 Автоматический парсинг плейсхолдеров из документа (включая хедеры/футеры)
- 📝 Динамическая веб-форма на основе найденных полей
- ⬇️ Генерация и скачивание готового `.docx` с подставленными значениями
- 📋 История сгенерированных документов

## Стек

- **Frontend**: React + TypeScript + Tailwind CSS + shadcn/ui
- **Backend**: Express + Node.js
- **Генерация**: docxtemplater + pizzip
- **Загрузка файлов**: multer

## Запуск

```bash
npm install
npm run dev
```

Приложение запустится на `http://localhost:5000`

## Как использовать

1. Создайте `.docx` файл с плейсхолдерами, например:
   ```
   Договор №{{номер_договора}}
   г. {{город}}, {{дата}}
   {{заказчик}} и {{исполнитель}} заключили договор...
   ```
2. Загрузите файл через веб-интерфейс
3. Заполните автоматически сгенерированную форму
4. Нажмите «Создать документ» — готовый `.docx` скачается автоматически

## Сборка

```bash
npm run build
NODE_ENV=production node dist/index.cjs
```

## API

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/templates` | Список шаблонов |
| POST | `/api/templates` | Загрузка шаблона (multipart/form-data) |
| GET | `/api/templates/:id` | Получить шаблон |
| DELETE | `/api/templates/:id` | Удалить шаблон |
| POST | `/api/generate` | Генерация документа |
| GET | `/api/history` | История генерации |
